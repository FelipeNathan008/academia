# Projeto Academia

Projeto de gerenciamento de academia desenvolvido em **Laravel** com banco de dados **MySQL**. Permite gerenciar alunos, professores, modalidades, matrículas, mensalidades, horários e frequência.

---

## Tecnologias

- PHP 8.2.x  
- Laravel 10.x  
- MySQL  
- Composer  

---

## Sequencia para criar o projeto

Criar o projeto com laravel
```
composer create-project laravel/laravel .
```

Iniciar o projeto com laravel
```
php artisan serve
```

Acessar conteúdo padrão do Laravel
```
http://127.0.0.1:8000
```
